# Yarn commands and their descriptions.

yarn webcrack:build - Builds src
yarn webcrack:watch - Watches and builds src.
yarn webcrack:dev   - Watches and builds src and webcrack.config.js

yarn serve - Start the dev server.
yarn dev   - Start the dev server with webcrack:dev

yarn newBlogEntry - Make an empty blog entry

yarn deploy - Send the dist folder to the right github branch and directory.
