# Yarn commands and their descriptions.

yarn funkophile:build - Builds src
yarn funkophile:watch - Watches and builds src.
yarn funkophile:dev   - Watches and builds src and funkophile.config.js

yarn serve - Start the dev server.
yarn dev   - Start the dev server with funkophile:dev

yarn newBlogEntry - Make an empty blog entry

yarn deploy - Send the dist folder to the right github branch and directory.
